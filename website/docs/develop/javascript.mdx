## Browser bundles

We distribute several JavaScript files, each of which "bundles" different parts
of the system so that you can tailor it to your site without the need for a
custom build process.

Our browser bundles (with the `.js` filename extension) conform to the
[Universal Module Definition (UMD)][umd] spec, and—barring availability of
browser globals such as `window` and `document`—should run in both web browsers
and Node.js.
