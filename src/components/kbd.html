The <kbd> HTML element represents computer keyboard interactions. We use them to display keyboard navigation instructions, such as pressing Tab or Enter. These elements use display: inline-block, so you'll need to use horizontal margin utilities like mr-8 (right), ml-8 (left), or mx-8 (left and right) to adjust spacing with adjacent text or other elements.

  • Control + Enter to enter the dashboard
  • Tab or Arrow to move between visuals
  • Control + Right arrow to enter a visual or filter
  • Escape to exit a visual or dashboard
